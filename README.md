# ML Step Master

Pythonをある程度使える初心者が、機械学習を難易度別・段階別に学べるWebアプリです。

## Features

- 難易度別の学習ロードマップ
- 各Levelの説明スライド
- 各Levelのコード演習問題
- 解答例と簡易チェック
- localStorageによる進捗保存
- ダークモード対応

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- CodeMirror

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production Build

```bash
npm run build
npm run start
```
